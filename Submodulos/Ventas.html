Ventas

Este sistema nos permite registrar la ventas que se realizan, de una manera más precisa para llevar un mejor control financiero.

Usuarios del subsistema

  • Gerente de operaciones
  • Recepcionista

Requerimientos por usuario

Requerimientos del Recepcionista

El Recepcionista debe ser capaz de:

  1. Realizar acciones sobre las ventas:
    • Resgitrar las ventas.
    • Actualizar los datos de las ventas
    • Consultar la información de cada venta
  2. Realizar corte de caja

El Gerente de operaciones debe ser capaz de:

  1. Realizar acciones sobre las ventas:
    • Resgitrar las ventas.
    • Actualizar los datos de las ventas
    • Consultar la información de cada venta
    • Eliminar una venta venta
  2. Supervisar que se realicen de manera correcta las ventas
  3. Realizar análisis de las ventas en el mes
