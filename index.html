📊 Sistema Motel - Módulos

1. Módulo de Promociones

Objetivo: Gestionar ofertas, descuentos y paquetes especiales para atraer clientes y aumentar ventas.

Datos del CRUD:

  • ID_Promocion, Nombre, Código, Tipo, Estado
  • Fecha Inicio/Fin, Días promoción, Horario promoción, Límites de uso
  • Condiciones, ID_Servicio, Monto mínimo, Membresía
  • Valor descuento (porcentaje/monto), Servicios incluidos

Encargado: Gerente de Marketing/Ventas o Gerente General

2. Módulo de Servicios

Objetivo: Administrar catálogo de servicios adicionales, precios y disponibilidad.

Datos del CRUD:

  • ID_Servicio, Nombre, Categoría
  • Descripción, Precio venta, Costo adquisición, Impuestos
  • Stock, Proveedor, Horario

Encargado: Gerente de Operaciones o Jefe de Recepción

3. Módulo de Membresía

Objetivo: Fidelizar clientes frecuentes mediante programa de lealtad.

Datos del CRUD:

  • ID, Nombre, Apellido Paterno, Apellido Materno, Fecha Nac., Correo, Teléfono
  • Nivel, Estado, Vencimiento, Puntos
  • Visitas, Gasto, Frecuencia de visita, Promociones Canjeadas
  • Beneficios y condiciones

Encargado: Recepcionista y Gerente de Marketing

4. Módulo de Ventas

Objetivo: Registrar transacciones del motel con control financiero y facturación.

Datos del CRUD:

  • ID_Venta, Fecha de venta, Hora de venta, Empleado, Cliente
  • Producto/Servicio, Cantidad, Precio, Promoción, Descuento
  • Subtotal, Impuestos, Total, Método de pago

Encargado: Recepcionista/Cajero, Gerente de Operaciones y Contador
